# Patient Disease Prediction App

## Overview

The **Patient Disease Prediction App** is an AI-powered healthcare platform designed to help patients understand potential health risks based on their symptoms, while providing doctors with advanced dashboards for monitoring and interaction. The system integrates predictive modeling, a supportive chatbot assistant, and secure blockchain-based data storage to ensure transparency, accuracy, and trust.

---

## Key Features

### AI-Driven Health Risk Predictions

* Uses machine learning and predictive modeling to analyze patient symptoms and health profiles.
* Generates an **initial positive/negative risk assessment** for potential conditions.
* Designed as a **preliminary screening tool**, encouraging patients to seek professional medical evaluation.

### Separate Dashboards for Doctors and Patients

#### **Doctor Dashboard**

* View patient profiles, prediction results, and symptom reports.
* Track patient histories securely stored on blockchain.
* Communicate with patients and provide follow-up recommendations.

#### **Patient Dashboard**

* Submit symptoms using an intuitive interface.
* View AI-generated predictions and doctor feedback.
* Access conversation history with the chatbot and clinicians.

### Built-in Chatbot Assistant

* Guides patients through symptom input.
* Explains prediction outcomes in easy-to-understand language.
* Encourages medical consultation when necessary.
* Acts as a supportive layer between the user and the predictive model.

### Blockchain-Based Health Records

* All chatbot interactions, predictions, and doctor-patient communication are securely stored using blockchain technology.
* Provides immutability, transparency, and trust.
* Ensures patient data integrity and traceability.

## Usage

1. Launch the app and sign in as either **patient** or **doctor**, or create an account as necessary.
2. Patients can input symptoms directly or through the chatbot.
3. The AI model processes the input and returns a risk prediction.
4. Doctors can review predictions, validate results, and update patient notes.
5. All interactions and results are securely logged on the blockchain.
